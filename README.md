## 📥 How to Download as PDF

### **Option 1: GitHub's Built-in Print (EASIEST)**
1. Go to any `.md` file
2. Click the **Print icon** (or press `Ctrl+P` / `Cmd+P`)
3. Select **Save as PDF**
4. Download

### **Option 2: Using GitHub Raw + Online Tool**
1. Right-click the file → **Open raw file**
2. Copy the URL
3. Visit: https://md-to-pdf.herokuapp.com/
4. Paste URL and convert
5. Download PDF

### **Option 3: Desktop Markdown Editor**
- Use [Typora](https://typora.io/) - Export as PDF
- Use [Markdownify](https://markdownify.js.org/)
- Use [Dillinger.io](https://dillinger.io/) - Export as PDF

### **Option 4: Direct Download via GitHub CLI**
```bash
gh repo clone kakdekaran/notes
# Then use any markdown to PDF converter
```
